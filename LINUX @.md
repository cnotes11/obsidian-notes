# LINUX Important Notes

**Linux Basics (Full Overview)**

---

## **1. What is Linux?**

- Linux is an **open-source operating system kernel**.
- Kernel = the “heart” of the OS → it manages memory, CPU, devices, and processes.
- When combined with software tools (GNU utilities), it becomes a **Linux distribution** (Ubuntu, Kali, Fedora, etc.).

---

## **2. Difference Between Linux & Unix**

|Feature|**UNIX**|**LINUX**|
|---|---|---|
|Year|1969|1991|
|Creator|Ken Thompson, Dennis Ritchie (AT&T Bell Labs)|Linus Torvalds (student, Finland)|
|Code|Closed-source (proprietary)|Open-source (GPL license)|
|Platforms|Mainly servers, workstations|Everywhere (PCs, servers, mobiles, IoT, supercomputers)|
|Cost|Paid|Free|
|Examples|Solaris, AIX, HP-UX|Ubuntu, Kali, Fedora, Debian, CentOS|

---

## **3. Who Made Linux & When?**

- **Linus Torvalds** → announced Linux in **August 25, 1991**.
- Started as a personal project to make a free alternative to MINIX.
- Later combined with **GNU tools** (developed by Richard Stallman’s GNU Project), creating a complete operating system.

---

## **4. Purpose of Linux**

- To provide a **free, open-source** alternative to UNIX.
- To be **portable** → run on any hardware.
- To be **secure** → multi-user, permission-based access.
- To support **servers, programming, hacking, mobile OS**.
- To encourage **community collaboration**.

---

## **5. Key Features of Linux**

1. **Open Source** – free to use and modify.
2. **Multi-user** – many users can work on the same system.
3. **Multitasking** – run many programs at once.
4. **Security** – user permissions, firewalls, encryption.
5. **Portability** – runs on PCs, servers, Android, IoT, supercomputers.
6. **Stability** – systems can run for years without reboot.
7. **Shell/Command Line** – powerful terminal commands.

---

## **6. Where is Linux Used?**

- **Servers** → Most websites (Google, Facebook, Amazon run on Linux).
- **Cybersecurity** → Kali Linux for hacking & penetration testing.
- **Mobile OS** → Android is based on Linux kernel.
- **Supercomputers** → 100% of top 500 supercomputers run Linux.
- **IoT Devices** → Smart TVs, routers, cars.
- **Cloud Computing** → AWS, Azure, Google Cloud use Linux.

---

## **7. Linux Timeline (Quick Look)**

- **1969** → UNIX created at Bell Labs.
- **1983** → GNU Project (Richard Stallman) starts.
- **1991** → Linus Torvalds releases Linux kernel (v0.01).
- **1992** → Linux released under GNU GPL license (free forever).
- **2000s → Today** → Linux powers servers, supercomputers, mobiles, clouds.
